# SHOPING APP
shopping app can list items and a user can choose products and buy it.

### Features
- can list products
- User can select/choose a product
- Selected user items should be added to the cart.


### Pages
application main pages:
+ Home page
+ About page
+ Product page
+ Product detail page
+ Shopping cart
